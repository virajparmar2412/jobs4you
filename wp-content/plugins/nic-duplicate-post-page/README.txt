== Description ==

This plugin is the duplicate posts, pages and custom post types easily using single click. You can duplicate your pages, posts and custom post by just one click and It will be duplicated as a draft.


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `nic-duplicate-post-page.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
